Migracion desde escritorio

Importa los datos generados desde la version Electron/SQLite.

1. Exportar SQLite

Ejecuta el script local incluido para convertir `sanpro.db` a JSON.

node pwa-supabase/tools/export-sqlite.js "C:\ruta\sanpro.db"

2. Importar archivo

Selecciona el JSON exportado. Se agregaran clientes y cobradores a Supabase.
